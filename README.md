# jaspy

## Deployment
### Docker
Install [Docker](https://docs.docker.com/install/) and
[docker-compose](https://docs.docker.com/compose/install/)
##### Clone repo
```
git clone https://github.com/jaspy2/jaspy.git
```
##### Modify config
```
cd jaspy
$EDITOR jaspy.env
```
##### Run compose
```
docker-compose up -d  
```
